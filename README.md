# Asif's Portfolio Website

A modern, responsive portfolio website showcasing AI/ML projects and skills.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Dark Theme**: Professional dark mode with cyan accent colors
- **Interactive Animations**: Smooth scrolling, hover effects, and scroll reveal animations
- **Modern UI**: Clean, minimal design with gradient effects
- **Sticky Navigation**: Fixed navbar with mobile hamburger menu
- **Project Showcase**: Interactive project cards with enhanced hover animations
- **Skills Display**: Animated progress bars and skill cards
- **Contact Form**: Frontend contact form with validation

## Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
- Modern CSS animations and transitions

## Sections

1. **Hero** - Introduction with name, title, and CTA buttons
2. **About** - Background and career goals
3. **Skills** - Technical skills with progress indicators
4. **Projects** - Portfolio of AI/ML projects
5. **Contact** - Social links and contact form

## Local Development

Simply open `index.html` in your browser to view the website locally.
